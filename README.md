# Coding Style
Use 2 spacing instead of 4<br>
Comment as you go<br>

*Do:*
```
<!--This Will Be Used For ...-->
<div>
  <h1>Hello World</h1>
</div>
```

*Dont:*
```
<div>
    <h1>Hello World</h1>
</div>
```

*Run Live Server:*
```
bun dev
```
